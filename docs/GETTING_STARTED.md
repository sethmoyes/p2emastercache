# Getting Started with Dungeon Turn V2

**Current Version:** 2.1.0 (February 2026)

## 5-Minute Quick Start

### 1. Generate Encounters (2 seconds)
```bash
python3 bin/generators/generate_dungeon_turn_v2.py --level 4 --floors 3
```

### 2. Read GM Guide (5 minutes)
Open: `gm/DUNGEON_TURN_V2_QUICK_REFERENCE.md`

### 3. Give Players Their Guide (5 minutes)
Give them: `players/DUNGEON_EXPLORATION_GUIDE.md`

### 4. Run Your Game!
- Track time (10 min = 1 die)
- Roll at 5 dice
- Look up sum in `gm/dungeon_turn_encounters_v2.md`
- Have fun!

---

## What You Get

### 288 Unique Encounters
- 96 events per floor (sums 5-100)
- 5 different categories
- Multiple solutions to every problem
- Living dungeon ecology

### Event Types
- **OPPORTUNITIES (22%)** - Reward creativity
- **COMPLICATIONS (21%)** - Force skill use
- **DILEMMAS (21%)** - Meaningful choices
- **ACTIVE THREATS (21%)** - Immediate danger
- **COMBAT (16%)** - Living dungeon

### Result
**Dungeons are FUN instead of tedious!**

### New in 2.1.0
- **Prepared Tokens:** Strategic time investment system
- **Enhanced Webapp:** Collapsible filters, better event display, interactive Otari map
- **Floor Filtering:** Fixed and accurate
- **Specific Creatures:** Every encounter has exact creatures with stats

## The Problem We Solved

**Before:**
- Players exhausted and bored
- Only using Detect Magic and Listen
- Closing doors on monsters
- No time pressure
- Tedious: detect, search, fight, repeat

**After:**
- Players engaged and creative
- Using ALL their skills
- Making meaningful choices
- Time pressure matters
- Fun: explore, choose, engage, succeed!

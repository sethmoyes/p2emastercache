# Wayne Reynolds Art Directory

This directory should contain Wayne Reynolds Pathfinder art for the background rotation.

## Required Images

Add these 5 images (JPEG format recommended):

1. `wr_goblin.jpg` - Goblin/small creature art
2. `wr_dragon.jpg` - Dragon or large monster art
3. `wr_dungeon.jpg` - Dungeon environment art
4. `wr_combat.jpg` - Combat scene art
5. `wr_monster.jpg` - Monster/creature art

## Where to Find Art

- Pathfinder Core Rulebook
- Pathfinder Bestiary books
- Paizo blog posts
- Official Pathfinder art galleries

## Image Specifications

- Format: JPEG (.jpg)
- Recommended size: 1920x1080 or larger
- Quality: High resolution for blur effect
- Aspect ratio: 16:9 preferred
